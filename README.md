# AGI Job Portal - Demo MVP

A demo-ready job portal for Gulf countries and South Asia, similar to GulfTalent and Indeed.

## Features

- **Job Search**: Keyword and location-based search with advanced filters
- **Multi-role Support**: Job Seeker, Employer, and Admin roles
- **Job Management**: Post, approve, and manage job listings
- **Application System**: Apply to jobs, save favorites, track applications
- **Admin Panel**: Approve jobs, verify employers, manage content

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- TypeScript
- Mock Data (No real backend)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
/app              - Next.js App Router pages
/components       - Reusable React components
/lib              - Utilities and mock data
/context          - React context providers
/types            - TypeScript type definitions
```
