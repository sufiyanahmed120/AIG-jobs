# Images Folder

Place your background image here.

## Hero Background Image

**File name:** `hero-background.jpg` (or `.png`, `.webp`)

**Recommended specifications:**
- Format: JPG, PNG, or WebP
- Size: 1920x1080px or larger (for high-resolution displays)
- File size: Optimize to under 500KB for faster loading
- Aspect ratio: 16:9 or wider

**How to add:**
1. Save your cityscape/Dubai skyline image
2. Name it `hero-background.jpg` (or keep your original name)
3. Place it in this folder: `public/images/`
4. Update the filename in `app/page.tsx` if you use a different name

The image will be automatically available at: `/images/hero-background.jpg`
