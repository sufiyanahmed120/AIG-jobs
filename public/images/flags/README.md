# Country Flag Images

Place your country flag image files in this directory.

## Required Files

Add the following flag image files (PNG, JPG, or SVG format recommended):

- `uae.png` - United Arab Emirates flag
- `saudi-arabia.png` - Saudi Arabia flag
- `qatar.png` - Qatar flag
- `kuwait.png` - Kuwait flag
- `oman.png` - Oman flag
- `bahrain.png` - Bahrain flag
- `pakistan.png` - Pakistan flag
- `india.png` - India flag

## Image Specifications

- **Recommended size**: 32x24 pixels (or 2:3 aspect ratio)
- **Format**: PNG (with transparency) or JPG
- **File naming**: Use lowercase with hyphens (e.g., `saudi-arabia.png`)

## Where to Get Flag Images

You can download flag images from:
- https://flagcdn.com/
- https://www.countryflags.com/
- https://flagpedia.net/

Make sure to use images that are free to use or properly licensed.
